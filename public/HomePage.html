How to use

A user needs to input "start order" e.g. such as AEDB, "number of turns" e.g. A, E, D, B (The letters represent the numerical number of the rider, such as A is rider1 and B is rider2) and "CDA scaling" (e.g. 0.98, 0.7, 0.6, 0.62). Next, The user needs to click the "Rider Profiles" button to input rider attributes. And the user can click the "Advanced Settings" to change default values of constants. Then, the user must select 5 riders from the drop down selection boxes. Finally, the user can click the "Calculate" button to get result.
